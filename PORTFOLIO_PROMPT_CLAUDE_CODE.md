## 🎨 DESIGN & ESTILO

- **Paleta de Cores:**
  - Primária: Azul profissional (#1e3a8a ou similar)
  - Secundária: Verde/teal para destaques
  - Neutros: Cinza escuro para texto, branco para fundo
  
- **Tipografia:**
  - Títulos: Inter, Poppins ou similar (sans-serif moderno)
  - Corpo: Segoe UI, -apple-system, sans-serif
  
- **Responsividade:**
  - Mobile-first design
  - Funcionamento perfeito em: Mobile (320px+), Tablet (768px+), Desktop (1024px+)

- **Animações:**
  - Scroll suave
  - Fade-in ao carregar seções
  - Hover effects nos cards
  - Transições suaves entre páginas
